Bonjour,

Je suis développeur web. Passionné depuis toujours d’informatique j’ai suivi une formation dans le but de réaliser ce métier.

J’ai ainsi appris l’utilisation et l’intégration de plusieurs languages web, l’utilisation d’outil afin d’améliorer l’accessibilité sur un site et enfin comment référencer au mieux les projets sur les différents moteurs de recherche.

En possession d’un diplôme bac+2 développeur web, je souhaite mettre mes compétences à contribution pour aider dans la création de sites web.

Je fais et ferai le nécessaire pour répondre à vos envies. Je mets mes compétences à votre service, pour créer le site web qui vous correspond, de la création du site à l'optimisation du référencement, jusqu'à l'entretien de ce dernier.

Je travaille essentiellement sur les Technologies HTML, CSS, Javascript, MySQL et NodeJs.

J'ai eu l'occasion de travailler avec beaucoup d'autres technologies, alors n'hésitez pas à me faire part de vos projets.
